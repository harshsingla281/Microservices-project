# Microservices Backend (Flask)

## Description
This project demonstrates a simple microservice architecture using Flask.

## Services

### Customer Service
GET /customers/<id>/orders

### Order Service
PUT /orders/<id>

## Tech Stack
- Python
- Flask
- Postman

## How to Run

Customer Service:
cd customer_service
python3 app.py

Order Service:
cd order_service
python3 app.py

## Testing
Use Postman to test APIs.
